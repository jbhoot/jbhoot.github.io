One way to remember box-sizing

17th September 2022

When box-sizing is set to box-sizing: content-box for an element, then width: 10rem applies only to the content of the element. The width of padding and border are added on top of width: 10rem.

When box-sizing is set to box-sizing: border-box for an element, then width: 10rem applies to everything within the borders of the element, including the borders. So, the width of content, padding, and the border counted within the width: 10rem.
