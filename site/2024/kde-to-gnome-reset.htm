1. Evaluate the need for ~/.config/{gtkrc,gtk-3.0,gtk-4.0,gtkrc-2.0} and any other besides them. These are either auto-generated or tainted by KDE.

  2. HiDPI: If you had configured scaling in KDE, then fonts may look blown up and out of proportion in relation to the interface in GNOME. Go to Tweaks -> Fonts -> Scaling Factor to 1.0. You should end up with Display Settings -> Scale == 200%, Fonts -> Scaling Factor == 1.0

  3. Reset several values in dconf. One among them is cursor-size and cursor-theme which may have been set to breeze.
